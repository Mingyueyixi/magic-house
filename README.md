## 介绍

这是一个用于改变一些应用功能的模块。

在这个应用中，为其他应用提供行为变更的能力称为“魔法”。

目前存在以下“魔法”：

**禁止**

- 对话框：对应用的对话框进行屏蔽，例如更新对话框。仅对原生应用生效，web/flutter/游戏等应用不生效

- 震动器：禁止应用的震动

**位置**

- AMap：修改使用了高德定位sdk的应用的位置信息

**开发工具**

日志捕捉：在手机上查看logcat（需root权限）

**其他**

- 屏幕旋转：自定义应用的旋转方式，如固定竖屏、强制旋转
